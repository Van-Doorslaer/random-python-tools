This script will help to convert PaVE names to an easier format.
The input can be 
	1) a fasta formated file (raw sequences or aliogned)
	2) a phylogenetic tree in newick fornat
	
The script requires ete3 

install ete3 by
pip install ete3



run as python rename_PaVE.py <file><type>
#
<file> = filename
<type> = fasta, newick
